Main Page

For our analysis we have collected and recoded the collected data to create a set of 32 variables.

In the initial analysis we were trying to predict the absolute money raised by the campaigns based on the features but it was ambitious and not accurate because of the high dimensionality of the data, outliers in each feature (e.g. Maximum Perk= 4Mn for a $5000 goal project) and the small size of the total population.

Therefore we decided to reduce the dimensionality by using regression models with penalty functions and used LARS and LASSO methods in cross-folds.

In the next stage we tried to create a multi-level logistic regression based classification system in three classes for Successful, Moderately Successful and Unsuccessful.
