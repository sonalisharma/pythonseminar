This GUI is used to read the images from the internet using url of the image and then provides mechanism to carry out various
actions on the image.

Note:
1) url should be a url pointing to an image, if the url or the image is invalid then the appropriate message is displayed in
comments box
2) The code saves temp image file, so while unning the code make sure the user has write access on the directory where the
code is being run from.

Methods used:
Following image transformations have been done:
1) Resize, made the image smaller
2) Enhance , enhanced the contrast of the image
3) Rotate , rotated the image by 45 degrees
4) Show Original, this is to show the original image
