HW8
==================

To run this, follow the steps below:
1) Download the folder.
2) On your terminal window, within the folder, run mywebsite.py
$ python mywebsite.py
3) This would fire the flask server and the page will open on localhost port 5000

Prerequistes:
1) SQL-alchemy
2) Flask
3) Pybtex
4) flask.ext.sqlalchemy
5) werkzeug (should come with flask)
